Wordelper?

This is a small project that I did back when Wordle became popular. It is an approach to get efficient words to find out the correct answer. As most of my projects, it is done in C++.

Initially, I just did it using a basic cmd window. It was extremely simple. The settings were stored on a file in the main folder. In there you would select which language did you want to use and how many characters you wanted the words to be. To tell the program the result of a word you would have to enter a number 0/1/2 for each character where 0 meant incorrect character (gray), 1 meant correct character (yellow) but on a wrong position and 2 meant correct character on the correct position (green).

 

Over a year later I found out about this project again and decided to make it look cooler, so I implemented the whole UI using Dear ImGUI by ocornut. This way, now the user has a proper scrollable list with many good words to use and it even includes a working simulation of the game. Just type the word you selected, click on the respective boxes to change their color and when you are done just submit! The settings were moved to the UI too, so no more need to open and edit weird files!

 

The premise is quite simple. First of all, we need a dictionary and a letter frequency for each language we want to be able to solve the Wordle game. Both of those can be found on the internet by doing some research. In the project I provided both files for English, Spanish and Catalan. For simplicity, I multiplied all values from the letter frequencies by 10 and rounded them to not have to handle with float values. Once this is set, we can start with the game.

For each word that fits the length that has been assigned in the settings, the program will assign it a score based on all the characters that conform the word using the letter frequency. For example, on a basic Wordle game (English, 5 characters) the best opening word with this system is atone with a score of 442. That is the sum of the letter frequency score in English: a(82), t(91), o(75), n(67), e(127). The program shows the list of the 200 best words sorted to the user, so they can grab the best word they think it can suit. Then the user has to type which word they actually selected and the result of it. Based on that, the list of available words will be updated, removing all the words with the banned characters and characters at wrong spots. After that, the process just repeats until the Wordle game finishes.
